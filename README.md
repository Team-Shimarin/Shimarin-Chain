### きらりちゃん可愛い

## using docker-compose
```
$ docker-compose up
```

run 4 `anzu-node` and  1 `kirarin-shower`.

`kirarin-shower` never have `any permissions` about `anzu-chain`.  
So, `anzu-chain` is not centralized.
