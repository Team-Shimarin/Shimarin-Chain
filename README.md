## using docker-compose
```
$ docker-compose up
```

run 4 `shimarin-node` and  1 `ena(斎藤さん)`.

`ena(斎藤さん)` never have `any permissions` about `shimarin-chain`.  
So, `shimarin-chain` is not centralized.
